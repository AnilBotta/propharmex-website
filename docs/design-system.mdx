# Propharmex Design System

This is the source-of-truth document for every visual and interactive decision on propharmex.com. All tokens live in a single file — `packages/config/design-tokens.css` — and all components live in `packages/ui/components/`. Every page, CMS block, and AI tool consumes this library; nothing is re-implemented.

**Consumer import pattern:**

```tsx
import { Button, Card, Dialog, MOTION, cn } from "@propharmex/ui";
import "@propharmex/config/design-tokens.css"; // once, in app/globals.css
```

---

## 1. Design principles

1. **Anti-hype.** Regulatory precision over marketing gloss. No sentence should be at home on a generic SaaS landing page.
2. **Evidence visible.** Numbers in JetBrains Mono, certifications in badges, "as-of" dates on every claim.
3. **Motion serves meaning.** Entries reveal hierarchy; exits dismiss. Never decorative; always honors `prefers-reduced-motion`.
4. **Accessibility is a gate, not a polish step.** Every interactive component passes WCAG 2.1 AA keyboard + screen-reader tests before merge.
5. **Density with breathing room.** Pharma buyers read. Give them dense technical detail inside generous whitespace.

## 2. Color tokens

All colors are exposed as `--color-*` custom properties on `@theme`, so Tailwind generates utility classes automatically (`bg-primary`, `text-accent-fg`, `border-border`, etc.).

### 2.1 Primary — Deep Teal

The spine of the brand. Use `--color-primary` (600) for primary surfaces, links, and focus rings; 50–200 for tinted backgrounds and 700–950 for text on tinted surfaces.

| Token | Hex / OKLCH | Contrast vs. `--color-bg` (#FAFAF7) | Usage |
|---|---|---|---|
| `--color-primary-50`  | oklch(97% 0.012 200) | 1.06:1 | Tinted section background |
| `--color-primary-100` | oklch(93% 0.020 200) | 1.27:1 | Hover on tinted surface |
| `--color-primary-500` | oklch(52% 0.082 200) | 4.9:1 | Muted accent text |
| **`--color-primary-600`** | **#0E4C5A** | **11.2:1 (AAA)** | **Brand base, primary button, links** |
| `--color-primary-700` | oklch(32% 0.064 200) | 10.8:1 | Hover on primary button |
| `--color-primary-900` | oklch(20% 0.040 200) | 16.4:1 | Headings on white |

### 2.2 Accent — Warm Amber

**Accent-only.** Never body text on white (contrast fails). Use for pills, stat highlights, callouts, and CTA moments that need to punch out from teal.

| Token | Hex | Usage |
|---|---|---|
| `--color-accent-500` | **#C99A4B** | Accent fills (buttons, badges, highlights) |
| `--color-accent-fg`  | oklch(18% 0.030 75) | Text on accent fill |

### 2.3 Neutrals — Pharma White + Slate

| Token | Value | Usage |
|---|---|---|
| `--color-bg`      | #FAFAF7 | Page background (Pharma White) |
| `--color-surface` | #FFFFFF | Card surface |
| `--color-fg`      | oklch(18% 0.010 260) | Body text |
| `--color-muted`   | oklch(54% 0.010 260) | Supporting / meta text |
| `--color-border`  | oklch(90% 0.006 260) | Dividers, input borders |

### 2.4 Semantic

| Token | Hex | Usage |
|---|---|---|
| `--color-success` | #16785A | Positive states, compliance confirmations |
| `--color-warn`    | #B8860B | Caveats, regulatory notes |
| `--color-danger`  | #A23B3B | Destructive / invalid states |
| `--color-info`    | #0E4C5A | Neutral informational callouts |

## 3. Typography

Three families, loaded via `next/font/google` and exposed as CSS variables:

| Token | Family | Role |
|---|---|---|
| `--font-display` | Manrope | Display, page titles, component titles, UI labels |
| `--font-body`    | Inter Tight | Body copy, long-form prose |
| `--font-mono`    | JetBrains Mono | Numeric values, stats, code, datestamps |

> **Substitution note.** The master plan references "JJT Mono" — treated as a typo for JetBrains Mono (documented in `docs/alignment-summary.md`).

### 3.1 Type scale (Tailwind classes)

| Use | Class | Computed |
|---|---|---|
| Display hero     | `text-5xl md:text-6xl font-display font-semibold tracking-tight` | 48–60 px |
| Page title (H1)  | `text-4xl font-display font-semibold tracking-tight` | 36 px |
| Section title (H2) | `text-3xl font-display font-semibold` | 30 px |
| Subsection (H3)  | `text-xl font-display font-semibold` | 20 px |
| Body             | `text-base font-body leading-relaxed` | 16 px / 1.625 |
| Meta / caption   | `text-sm text-muted` | 14 px |
| Stat value       | `text-4xl font-mono tabular-nums` | 36 px |

## 7. Motion

Motion tokens live in both CSS (`--duration-*`, `--ease-*`) and TypeScript (`MOTION` export from `@propharmex/ui`). Framer Motion components must import from TS; CSS transitions use the CSS values.

| Token | Value | Purpose |
|---|---|---|
| `--duration-enter`   | 240 ms | Entries (fade-in, slide-up) |
| `--duration-exit`    | 180 ms | Exits (fade-out, slide-down) |
| `--duration-stagger` | 40 ms  | Sibling stagger |
| `--ease-out` | `cubic-bezier(0.16, 1, 0.3, 1)` | Entries |
| `--ease-in`  | `cubic-bezier(0.7, 0, 0.84, 0)` | Exits |

### 7.1 Reduced motion contract

Every Framer Motion block wraps its variants in `useReducedMotion()` from `@propharmex/ui`. When true, animation distance collapses to 0 and durations collapse to `0.01ms` via the global `@media (prefers-reduced-motion: reduce)` block in `design-tokens.css`.

```tsx
import { motion } from "framer-motion";
import { useReducedMotion, MOTION } from "@propharmex/ui";

function Reveal({ children }: { children: React.ReactNode }) {
  const reduced = useReducedMotion();
  return (
    <motion.div
      initial={reduced ? false : { opacity: 0, y: 12 }}
      whileInView={{ opacity: 1, y: 0 }}
      viewport={MOTION.viewport}
      transition={MOTION.enter}
    >
      {children}
    </motion.div>
  );
}
```

## 8. Accessibility contract

Every interactive component in `@propharmex/ui` meets the following — enforced by `design:accessibility-review` before any PR merges.

- **Keyboard.** Full tab order, `Enter`/`Space` to activate, `Escape` to dismiss, arrow-key navigation where a roving tabindex is appropriate (Tabs, Select, Accordion triggers as group).
- **Focus.** Visible `focus-visible:ring-2 focus-visible:ring-primary focus-visible:ring-offset-2` on every interactive element. Global fallback in `design-tokens.css`.
- **ARIA.** Radix primitives supply it by default (Dialog, Tabs, Accordion, Select, Tooltip). Hand-rolled components (Breadcrumb, Pagination, Callout) declare `role`, `aria-label`, `aria-current`, `aria-live` as appropriate.
- **Touch targets.** All interactive elements ≥44×44 px at `sm` size (`min-h-11 min-w-11`).
- **Contrast.** AA (4.5:1) minimum on text; AAA (7:1) on primary flow text. Deep Teal #0E4C5A on Pharma White = 11.2:1.
- **Reduced motion.** Honored at CSS, Tailwind (`motion-reduce:` classes), and Framer levels.
